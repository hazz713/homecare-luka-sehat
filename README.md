# Homecare Luka Sehat

Website profil layanan Homecare Luka Sehat.

## Isi project

- `index.html` untuk halaman utama
- `styles.css` untuk tampilan
- `script.js` untuk animasi ringan
- `assets/` untuk logo dan aset visual

## Cara jalankan

Buka `index.html` langsung di browser.

## Publish ke GitHub Pages

1. Upload semua file project ke repository GitHub.
2. Buka `Settings` repository.
3. Masuk ke `Pages`.
4. Pilih `Deploy from a branch`.
5. Pilih branch `main` dan folder `/root`.
6. Simpan lalu tunggu link website aktif.
